# 插件
| 插件              | 功能              | 快捷键                                             |

| mini.ai           | a和i的拓展        | g[<br>g]                                          |
| mini.align        | 对齐              | ga                                                 |
| mini.operators    | 文本对象操作      | 计算g=<br>交换gX<br>复制gm<br>替换gr<br>排序gs |
| mini.pairs        | 自动补全引号括号                  |                                                    |
| mini.surround  | surround操作  | 添加sa<br>删除sd<br>替换sr<br>查找sf<br>高亮sh   |
|mini.comment|注释| gc|
|mini.bracketed|使用[]进行前进后退（buffer,treesitter...）|[b, ]b....| 	
|mini.clues|按键提示（更像helix）|key|
|mini.files|file-tree(oil.nvim)|<leader>mf|
